Create your webpage using this app

This app is a shortcut between home page and the QR scanner. It allows to open javascript-functions passing the QR code value.
Why an app? Simply, it doesn't ask always the access to your camera if you want scan a code. And you can get also GPS coorinates (future) directly to your webpage.
Another cool function is that you can scan when you are offline and insert the code once you have internet access.
